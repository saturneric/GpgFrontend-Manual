Other options, for special cases only:

- [**Mirror Site**](https://ftp.bktus.com/GpgFrontend/): the same files from another source, with no tracking or sign-in.

### Understanding the File Names

Use this table to find the right file for your system.

| Filename Pattern               | What It's For                                                    |
| ------------------------------ | ---------------------------------------------------------------- |
| `-windows-x86_64-portable.zip` | Leaves no trace on Windows. Run it straight from a USB drive.    |
| `-windows-x86_64.msix`         | Modern MSIX installer for Windows (Recommended).                 |
| `-windows-x86_64.msi`          | Standard MSI installer; made for enterprise setups.              |
| `-macos-<ver>.dmg`             | macOS disk image; `<ver>` shows the lowest macOS version needed. |
| `-linux-<arch>.AppImage`       | Standalone Linux app. Make it executable, then run it.           |
| `*.sig`                        | Signature file. Always use it to check the download is genuine.  |
| `-qt5-`                        | Older system build (e.g., Windows 7, Ubuntu 20.04).              |

<Aside type="note" title="Qt5 builds moved to the Mirror Site">
  Starting with **v2.2.1**, the Qt5 (`-qt5-`) builds are no longer on the GitHub
  releases page. They now live only on the [Mirror
  Site](https://ftp.bktus.com/GpgFrontend/). These builds are for older systems
  only. If your system is current, use the standard build instead.
</Aside>

    <Aside type="caution" title="About GpgFrontend Lite">
      **GpgFrontend Lite** is made for the strict macOS sandbox. It leaves out
      GnuPG and uses only the memory-safe **rPGP engine** for a smooth Mac App
      Store experience. If you want the full experience, including GnuPG support
      and advanced hardware token support, install the standard Desktop Edition
      via Homebrew or GitHub.

      **Back up your keys right away.** Lite stores your keys inside an Apple
      sandbox, so removing the app or resetting your system can erase them, and
      lost keys mean your data is gone for good.
    </Aside>
